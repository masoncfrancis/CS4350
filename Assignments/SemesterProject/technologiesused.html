Technologies Used


XHTML

Aspects of XHTML are used throughout the site, such as in the DOCTYPE tag at the beginning of every page which is specific to XHTML.

HTML5

Aspects of HTML5 are used throughout the site, such as in the video tag which is new in HTML5.

HTML5 Canvas Element

I added the SUUSA logo inside a canvas element at the top of the projects.php page. Just used for branding purposes.

HTML5 Video Element

The video is intended to be a screen recorded tutorial of how to use the site. Currently I have included a placeholder video for now.

CSS

My CSS properties are generally being used for positioning and size of elements. I used an external stylesheet for Bootstrap to style most of the elements and included my own to position and size things how I want. CSS is used on every page in the site.

JavaScript

JavaScript is used in the officialproc.php file to redirect the user back to the officials.php page after adding or editing an elected official.

Dynamic JavaScript

Dynamic JavaScript is used on the addofficial.php and editofficial.php pages to change and disable the save button after submitting the form.

PHP

PHP is used extensively, such as to add the dynamic navbar across the site, to create and modify database records, and to populate data on the officials.php and editofficial.php pages.

Database

The database in my project is primarily used to store information about elected officials and project information. I store the code used to connect to the database in a file called db.php. I use PHP in editofficial.php to populate the fields for the given official id that is passed via GET in the URL. In officials.php I also query the database to find what officials exist in the database and show them to the user. I communicate with the database in officialproc.php to update the record for the person, and to create new records.

SVG Logo

The T-Bird logo (which I do not claim to have created, it was created by SUU's marketing team) is on the left side of the navbar at the top of every page.

Web Server

The site runs on an Apache server that I obtained in a Bitnami WAMP Stack bundle.
